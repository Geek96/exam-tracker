# ADR-001: 使用原生 HTML/CSS/JS，不引入前端框架

**日期**: 2026-04-28  
**状态**: 已接受

## 背景

项目启动时需要选择技术栈。可选方案包括 React、Vue、Svelte 或原生 JS。

## 决策

使用纯 HTML + CSS + Vanilla JavaScript，不引入任何前端框架或打包工具。

## 理由

- **部署简单**：Vercel 可直接托管静态文件，无需构建步骤
- **依赖零风险**：无 `node_modules`，无版本冲突，无依赖漏洞
- **加载极快**：无框架运行时开销，直接加载文件
- **易于 AI 辅助开发**：无抽象层，AI Agent 可直接理解和修改代码
- **项目规模匹配**：当前功能集（~5个页面、~5个API）不需要框架带来的复杂度

## 代价

- 无组件化，代码复用依赖函数抽取
- 无状态管理库，数据流需手动维护
- 无类型安全（未使用 TypeScript）

## 未来预留

如果项目扩展到 10+ 页面或需要复杂状态，可以考虑引入 Vite + Vanilla TS 作为最小化升级路径，无需替换全部代码。
